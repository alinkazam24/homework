Задание — 

«Красивая форма с псевдоклассами и SVG-иконками» Цель: 

создать адаптивную, доступную и аккуратную форму регистрации, использовать CSS-псевдоклассы (
:focus,-
:hover,-
:invalid,-
:placeholder-shown,-
:checked
























































     и т.д.) и встроенные SVG-иконки. Требования: Использовать минимум 4 разных псевдокласса. Вставить SVG-иконки. Валидировать поля через HTML5 (required, type) и показать стиль ошибки через :invalid. Кнопка отправки меняет вид при :hover и :active. Добавить чекбокс (с использованием :checked) и плейсхолдер-анимацию с :placeholder-shown.
